# Pinecone Knowledge Indexer

Upload files (PDF, DOCX, TXT) → vector embeddings → semantic search in Pinecone.

## Usage

1. Set `PINECONE_API_KEY` in Space settings
2. Upload a file via the web UI
3. Search across your documents

Built with FastAPI + Pinecone + agy (Antigravity CLI)
